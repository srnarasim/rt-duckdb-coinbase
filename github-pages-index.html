About This Application

This application demonstrates real-time BTC/USD price analytics using different technologies:

  • D3 Version: Uses D3.js for visualization with simulated data. Works in all modern browsers.
  • WebAssembly Version: Uses Rust compiled to WebAssembly with DuckDB-WASM for in-browser analytics. Requires WebAssembly support.

Both versions provide real-time price charts and analytics for Bitcoin price data.

Troubleshooting

If you encounter issues with the WebAssembly version:

  • Make sure your browser supports WebAssembly
  • Try using the D3 version instead
  • Check the browser console for error messages
